⚔ **Top-Down Mini-Game** ⚔
![скриншот](https://i.postimg.cc/Vvh0tGqB/rumata.png)

Функционал:
	
	Процедурная генерация по тайлам на основне simplex noise
	
	Игрок с базовым управлением,анимациями,FSM и дешем с использованием шейдера для визуала
	
	Базовые мобы с behaviour и навигацией - Патрулирование, чейзинг игрока, деш в сторону игрока с перезарядкой.
	
	Сбор монет, магнит к игроку через tween.
	
	Простая катцена по завершению уровня
	
	UI.

	Система сохранений Dictionary to json.


На этом все, Спасибо за внимание!
